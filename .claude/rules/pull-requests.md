# Pull Request Rules

## AI Attribution

**NEVER add AI attribution footers to PR descriptions.**

- ❌ No `🤖 Generated with [Claude Code](https://claude.com/claude-code)`
- ❌ No `Co-Authored-By: Claude` or similar AI co-author lines
- ❌ No any other AI-generated attribution or signatures

Keep PR descriptions clean and human-authored in appearance.
